### **Unit Economics (The 30% Strategy)**

**1. Розрахунок для чеку 500 грн (Стрижка):**
- Чек: 500 грн
- Комісія платформи (30%): 150 грн
- Доход барбера: 350 грн

**2. Розрахунок для чеку 600 грн (Стрижка + Борода):**
- Чек: 600 грн
- Комісія платформи (30%): 180 грн
- Доход барбера: 420 грн

**3. Порівняння з салонами Рівного:**
У салонах барбер отримує лише 35-40% від чеку.
- Для чеку 500 грн у салоні: 175 – 200 грн
- У Safecut: 350 грн

Отже, барберу вигідніше працювати з нами, оскільки його дохід значно вищий.

### **Technical Constraints (Технічне ТЗ для розробників)**

**1. Правило для коду:**
- Послуги "beard" та "family" можуть бути замовлені лише коли обрана "haircut".
- Кодове впровадження:
  ```typescript
  // Перевірка доступності послуг
  if (selectedServices.includes('haircut')) {
    unlock 'beard' and 'family';
  }
  ```

**2. Правило для DistrictSelector:**
- Барбер бачить замовлення лише свого району.
- Кодове впровадження:
  ```typescript
  // Фільтрування за районом барбера
  const filteredOrders = orders.filter(order => order.district === barberDistrict);
  ```

### **Loyalty Program**

**Механіка лояльності:**
- Кожен клієнт отримує бонус після 10-ї стрижки.
- 10-та стрижка безкоштовна.

**Кодове впровадження:**
- Оновлення лічильника в Firebase після кожної стрижки.
